## Required Assets

3. **demo-multi-server.gif** (800px width)

### Tools for Creating GIFs
- **Screen Recording**: QuickTime, OBS, or ScreenToGif
- **GIF Conversion**: Gifski, ezgif.com, or ffmpeg
- **Optimization**: gifsicle or ezgif.com

### Recording Tips
1. Use a clean browser window (no extensions visible)
2. Set window width to 1600px for clarity
3. Use dark theme for better contrast
4. Keep mouse movements minimal and purposeful
5. Optimize final GIF to under 5MB

## Banner Image

The **banner.png** should be in `/public/banner.png` and include:
- NulaLabs logo
- Visual representation of workflow/data analysis
- Professional color scheme (#6D33A6, #1E2940, #265573, #0FDBF2)
- Dimensions: 1200x400px
- Support for light/dark modes (optional)

### Creating the Banner
Tools: Figma, Canva, Adobe Illustrator, or Inkscape

Design elements to include:
- Project name (NulaLabs)
- Tagline
- Visual elements: workflow graph, charts, AI assistant icon
- MCP logo/reference
